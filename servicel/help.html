Preguntas frecuentes sobre la aplicación Servicel para Android


Actualizado hace unos días

¿Que es Servicel?

Esta aplicación facilita la gestión de tus servicios de Nauta (www.nauta.cu), ofreciéndote información detallada sobre tus paquetes de datos, saldo y planes.

¿Servicel es o tiene algún vínculo con ETECSA?

No, Servicel es una aplicación de terceros, que sólo toma la información extraida desde el portal web www.nauta.cu para ser mostrada al usuario más detalladamente en una aplicación móvil.

¿Dónde puedo descargar e instalar la aplicación?

  • Play Store

  • Apklis

    • ¿En cuales sistemas operativos se puede usar Servicel?

    Actualmente damos soporte a dispositivos con sistema operativo:


  • Android OS 6.0 y versiones posteriores


    • Si su dispositivo es compatible instale Servicel e inicie sesión o registrese para comenzar a usar la Aplicación.

    ¿Cómo puedo verificar la versión de la aplicación?

    Para comprobar la versión de su aplicación, navegue a Configuración para ver la información de la versión.

    ¿Que pasa si no estoy registrado/a en el portal nauta?

    Servicel está diseñada para que funcione mediante www.nauta.cu, si usted no tiene una cuenta no podrá usar la Aplicación, para ello deberá registrarse.

    ¿Cómo me registro?

    Podrá hacerlo mediante www.nauta.cu o desde la propia aplicación, cuando inicia por primera vez le permitirá crear una cuenta.

    No puedo iniciar sesión, ¿que puedo hacer?

    Primero compruebe tener una buena conexión a internet, si tiene conexión, asegúrese de que el portal nauta no esté en mantenimiento. Si después de esto sigue teniendo problemas con el inicio de sesión pongase en contacto con nosotros.
